# 🌐 Personal Portfolio Website

A modern, premium dark-themed personal portfolio website built using HTML, CSS, and Vanilla JavaScript.

This portfolio showcases my skills, projects, education, and certifications with a clean glassmorphism design, smooth animations, and fully responsive layout.

## 🚀 Live Demo
🔗 https://uzaportfollio.netlify.app/

---

## ✨ Features

- Dark theme with glassmorphism UI
- Animated gradient and floating background effects
- Scroll reveal animations using Intersection Observer
- Responsive design (Mobile, Tablet, Desktop)
- Sticky navigation with active section highlight
- Typing animation in hero section
- Categorized skills section with animations
- Project showcase with live demo & GitHub links
- Contact form with JavaScript validation
- Scroll progress indicator
- Clean and modern UI/UX

---

## 🛠 Tech Stack

- HTML5
- CSS3 (Flexbox, Grid, Glassmorphism, Animations)
- Vanilla JavaScript
- Intersection Observer API
